```
│  ○ Active Directory             │

│                                 │
│  如果选择 Dataverse:             │
```

## 错误处理流程 (Error Handling Flow)

```
任何步骤发生错误
Error in any step
         │
         ▼
┌─────────────────────────────────┐
│    错误处理                     │
│    Error Handling               │
│                                 │
│  发布过程中发生错误：            │
│  Error occurred during          │
│  deployment: [错误信息]          │
│                                 │
│  ○ 确定                         │
└─────────────────────────────────┘
         │
         ▼
      流程结束
      Process ends
```

